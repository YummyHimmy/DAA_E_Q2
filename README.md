EXORCIZE is a grid-based horror strategy game developed using Python and the Pygame library as part of the Design and Analysis of Algorithms course project. The main focus of this project is the implementation of the Breadth-First Search (BFS) algorithm for enemy pathfinding inside a procedurally generated maze environment.
In the game, the player must collect several manuscripts while avoiding a ghost that continuously searches for the shortest reachable path toward the player. The game map is represented as an unweighted graph where each walkable tile acts as a node connected to neighboring nodes.
BFS was chosen because it is suitable for unweighted graph traversal and guarantees the shortest path solution. Besides implementing BFS, the project also includes procedural map generation, safe-zone mechanics, difficulty selection, movement limitation, and user interface systems.
Based on the testing results, BFS performs efficiently in small grid environments and can generate stable ghost movement during gameplay. This project demonstrates how graph traversal algorithms studied in class can be applied in an interactive game application.
